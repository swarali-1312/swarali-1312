## Projects

### [Transit Light Curve Analysis — Kepler-10b](https://github.com/swarali-1312/kepler10b-transit)
*Independent project · Python · lightkurve · NASA Kepler data*

Detected the transit of exoplanet Kepler-10b in real photometric data from the Kepler space telescope. Phase-folded and binned the light curve to extract a transit depth of ~1.86 × 10⁻⁴, yielding a planetary radius estimate of ~1.6 R⊕ — consistent with the known value for this rocky super-Earth.

**Skills:** `lightkurve` · `numpy` · `matplotlib` · transit photometry · phase-folding
